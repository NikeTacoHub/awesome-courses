# awesome-courses

> 🚧 站点积极建设中，敬请期待 🚧

各个学科的开源课程资料库，欢迎分享与贡献 / An open-source repository of course materials. Contributions welcome!

本仓库仅标注教材/课件/作业等课程资料，笔记/试题等请查看资料库内其他仓库。

上传非原创内容前请征得原作者授权，感谢理解。

贡献者精确到具体资源（笔记等其他资源作者将展示于其他仓库），仓库及作者按课程号以及时间顺序排序，若均相同则不区分顺序。若标注内容有误，请提交PR/Issue勘误，非常感谢！
